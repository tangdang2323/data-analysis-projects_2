# Анализ мобильной игры

## Описание проекта
**Цель:** оценить удержание игроков, выявить точки падения активности и рассчитать вирусный коэффициент.

## Использованные инструменты
- **SQL (PostgreSQL):** оконные функции, JOIN, CTE, агрегации
- **Excel / Google Sheets:** постобработка, визуализация
